# nodemcu-esp8266-jammer-project
DIY project based on NodeMCU ESP-12E (ESP8266MOD) with TP4056-TypeC charging module, rocker key control, and rechargeable 18650 Li-ion battery (2000mAh recommended with holder). Optional small high-efficiency fans (&lt;4cm) can be used for cooling. Compact, low-power, and practical design for IoT experiments.
